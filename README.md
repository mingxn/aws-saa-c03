# AWS SAA-C03 Question Bank

Interactive question bank for AWS Solutions Architect Associate (SAA-C03) exam preparation with **Practice Mode** and **Exam Mode**.

## Development

**Prerequisites:** [Hugo Extended](https://gohugo.io/installation/) (v0.100.0+)

**Run locally:**
```bash
hugo server --buildDrafts
```

Visit: http://localhost:1313/aws-saa-c03/

## Deployment

Deploys to GitHub Pages: https://mingxn.github.io/aws-saa-c03/
